开搞

当我们安装完 node 之后就可以进行愉快的后续构建工作了，在这里推荐大家使用官方的脚手架进行开发。

目前前端很多主流框架都有自己的官方脚手架来用于快速搭建项目，比如 Vue 的官方脚手架工具 vue-cli 、 React 等，这里我们就来介绍下如何利用 vue-cli 快速构建我们的项目。
